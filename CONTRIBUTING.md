# Contributing to Feza-X

Welcome to the Feza-X CubeSat Architecture project! We are excited to have you contribute.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Submit a Pull Request with a detailed description of your changes.

## Design Guidelines
- Ensure all architectural diagrams are in Mermaid format.
- Documentation should be in both Turkish and English (optional but preferred).
- Keep CAD designs modular.
